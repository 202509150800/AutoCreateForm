# 🚀 5 分鐘快速開始

## 只需這 5 步即可運行應用！

### 第 1 步：打開終端 (1 分鐘)

**Windows:**
- 按 `Win + R`
- 輸入 `cmd`
- 按 Enter

**macOS:**
- Command + Space
- 輸入 `terminal`
- 按 Enter

**Linux:**
- Ctrl + Alt + T

### 第 2 步：進入專案目錄 (1 分鐘)

複製並貼上此命令，然後按 Enter：

```bash
cd "c:\Users\fung\Desktop\2025\test\AutoCreateForm"
```

### 第 3 步：安裝依賴 (2 分鐘)

輸入並執行：
```bash
npm install
```

*等待完成。會看到類似 "added 150 packages" 的訊息*

### 第 4 步：啟動應用 (1 分鐘)

輸入並執行：
```bash
npm start
```

*應看到:*
```
✓ 伺服器運行在 http://localhost:3000
✓ 打開瀏覽器訪問: http://localhost:3000
```

### 第 5 步：打開瀏覽器

在瀏覽器網址列輸入並訪問：
```
http://localhost:3000
```

## 🎯 使用流程

### 步驟 1: 定義欄位

1. 在第一個區域「定義欄位」中輸入您要使用的欄位名稱
2. 例如：`姓名`、`電話`、`地址`、`model`、`MAAC_address` 等
3. ID 欄位已預設存在

### 步驟 2: 上傳 Excel

1. 上傳一個 Excel 檔案 (.xlsx 或 .xls)
2. **重要：** Excel 的列標題必須與您定義的欄位完全相同

**Excel 範例：**
```
| ID | model | MAAC_address | ... |
|----|-------|--------------|-----|
| 1  | A-100 | 台北市信義區  | ... |
| 2  | B-200 | 台中市西屯區  | ... |
```

### 步驟 3: 上傳 Word 範本

1. 準備一個 Word 檔案 (.docx)
2. **在要填寫的位置使用 `{欄位名稱}` 格式**

**Word 範例：**
```
產品型號: {model}
安裝地址: {MAAC_address}
客戶編號: {ID}
```

### 步驟 4: 設置並生成

1. 選擇是否生成 PDF
2. 點擊「開始生成」
3. 下載 ZIP 檔案包含所有生成的檔案

|------|--------|
