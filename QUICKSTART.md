# 快速入門指南

## 5 分鐘上手

### 1️⃣ 啟動應用
```bash
cd AutoCreateForm
npm install
npm start
```
打開瀏覽器訪問：http://localhost:3000

### 2️⃣ 定義欄位
在介面的「步驟 1」中：
- ID 欄位已預設（必須）
- 輸入其他欄位名稱，如：姓名、電話、地址
- 點擊「➕ 添加欄位」

### 3️⃣ 準備 Excel 資料
建立 Excel 檔案，格式如下：
```
ID  | 姓名 | 電話      | 地址
----|------|-----------|--------
1   | 張三 | 0912345678| 台北市
2   | 李四 | 0987654321| 台中市
```
儲存為 `.xlsx` 格式

### 4️⃣ 準備 Word 範本
建立 Word 檔案，在要填寫的位置使用佔位符：
```
姓名：{{姓名}}
電話：{{電話}}
地址：{{地址}}
```
儲存為 `.docx` 格式

### 5️⃣ 上傳並生成
1. 上傳 Excel 檔案
2. 上傳 Word 範本
3. 勾選生成選項（Word / PDF）
4. 點擊「開始生成」
5. 下載 ZIP 檔案

## 完成！🎉

生成的檔案會根據 ID 命名：
- `1.docx`, `1.pdf`
- `2.docx`, `2.pdf`
- ...

## 注意事項

- ✅ Excel 必須包含 ID 欄位
- ✅ 欄位名稱必須與佔位符完全相同
- ✅ 佔位符格式：`{欄位名稱}`（單括號）
- ⚠️ PDF 需要安裝 LibreOffice
