## 📱 按設備/格式

### **📖 Web 瀏覽**
- 所有 Markdown 文件都支持 GitHub 的 Web 預覽
- 建議: 在瀏覽器中查看

### **💻 代碼編輯器**
- 推薦使用 VS Code 打開文件
- 支持 Markdown 預覽和搜索

### **📱 手機**
- 不推薦在手機上查看代碼文檔
- 適合查看 README 和 FAQ

### **📰 打印**
- 某些文檔適合打印 (README, QUICKSTART, FAQ)
- 避免打印長文檔 (BEST_PRACTICES - 800+ 行)
